# assets/

Pasta reservada para os recursos visuais do jogo.

## Organização sugerida

```
assets/
├── cobras/      → sprites e ilustrações da Fase 1 (Cobra)
├── dragoes/     → sprites e ilustrações da Fase 2 (Dragão)
├── jacares/     → sprites e ilustrações da Fase 3 (Jacaré)
├── ui/          → ícones, botões, HUD elements
└── cenarios/    → backgrounds de cada fase
```

Formatos recomendados: `.png` (sprites com transparência), `.webp` (fotografias/backgrounds).
